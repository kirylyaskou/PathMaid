# Справочники

PathMaid включает полные справочники по контенту PF2e. Данные загружаются из Foundry VTT при синхронизации.

## Заклинания

Раздел **Заклинания** — поиск и просмотр всех заклинаний системы.

**Фильтры:**
- Ранг (1–10)
- Традиция (арканная, божественная, природная, мистическая)
- Школа (эвокация, иллюзия, некромантия и т.д.)
- Признаки (attack, healing, mental и т.д.)
- Источник пака

**Карточка заклинания** показывает:
- Ранг, традиции, школа, признаки
- Время каста, дальность, цель, длительность
- Описание с таблицами повышения (heighten)
- DC и бонус атаки — рассчитывается для уровня отряда из Главной страницы

## Предметы

Раздел **Предметы** — снаряжение, магические предметы, расходники.

**Фильтры:**
- Уровень
- Категория (оружие, броня, чудесный предмет, зелье и т.д.)
- Цена
- Объём

## Опасности

Раздел **Опасности** — ловушки, опасная среда, сложные опасности.

Каждая опасность показывает:
- Уровень, сложность (простая / сложная)
- Стелс DC, DC обнаружения
- HP, закалка, BT
- Действия / реакции
- Процедура обезвреживания

**Использование в встречах:** нажмите **Добавить во встречу** для включения в XP-расчёт.

## Действия

Раздел **Действия** — все базовые действия системы PF2e.

Включает:
- Основные боевые действия (Strike, Stride, Step и т.д.)
- Специальные действия (Grapple, Trip, Disarm и т.д.)
- Реакции (Attack of Opportunity, Shield Block и т.д.)
- Действия из подвигов

Каждое действие показывает стоимость (1A/2A/3A/R/F), триггер (для реакций), применяемую проверку, и степени успеха.

## Состояния

Раздел **Состояния** — полный справочник по состояниям PF2e.

Для каждого состояния:
- Описание эффекта
- Возможные значения (для числовых состояний)
- Взаимодействие с другими состояниями (иммунитеты, отмены, снижения)
- Источники получения

Кнопка **Применить к участнику** (если открыт бой) добавляет состояние в активный бой.

## Подвиги

Карточки подвигов доступны в статблоке существа при наличии подвигов в raw_json. Отдельная страница справочника подвигов не реализована в текущей версии.
