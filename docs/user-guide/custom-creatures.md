# Конструктор существ

PathMaid включает инструмент для создания homebrew существ по правилам построения существ PF2e (Gamemastery Guide, глава «Building Creatures»).

## Создание нового существа

1. Перейдите в **Конструктор** (иконка молотка)
2. Нажмите **Новое существо**
3. Выберите режим:
   - **Оригинальное** — создаётся с нуля
   - **На основе существующего** — берётся статблок из бестиария как отправная точка

## Базовые параметры

| Поле | Описание |
|---|---|
| Имя | Отображается в трекере боя |
| Уровень | -1 до 25, определяет эталонные значения статистик |
| Редкость | Обычный / необычный / редкий / уникальный |
| Размер | Крошечный → Колоссальный |
| Тип | Аберрация, зверь, конструкт, и т.д. |
| Признаки | Дополнительные признаки |

## Статистики

### Эталонные значения

При изменении уровня PathMaid подсвечивает эталонные значения согласно таблицам из Gamemastery Guide:

- **Экстремально** — для специализированного ролевого монстра
- **Высокое** — стандарт для основной роли
- **Умеренное** — стандарт для вторичной роли
- **Низкое** — слабая сторона
- **Ужасное** — намеренная уязвимость

Цвет поля показывает отклонение от эталона: зелёный (в норме) → жёлтый (отклонение) → красный (значительное отклонение).

### Роли

Применение роли автоматически расставляет статистики по шаблону:

| Роль | Описание |
|---|---|
| Брут | Высокий урон, HP; низкий AC |
| Снайпер | Высокая атака, умеренный урон |
| Лидер | Высокая воля, умеренные характеристики |
| Скрытный | Высокий Рефлекс, умеренный AC |
| Мистик | Высокая воля, умеренная атака |
| Солдат | Высокий AC, умеренное всё |

Применение роли — стартовая точка, все значения можно редактировать вручную.

## Атаки

Добавьте одну или несколько атак:

- **Имя** — название атаки (например, «Коготь», «Укус»)
- **Бонус атаки** — модификатор броска атаки
- **Кубики урона** — формула (например, `2d8+6`)
- **Тип урона** — рубящий / колющий / дробящий / огненный и т.д.
- **Признаки** — agile, reach, sweep и т.д.

## IWR (иммунитеты, уязвимости, сопротивления)

Добавьте иммунитеты, уязвимости и сопротивления через соответствующие секции. Для каждого элемента указывается тип и (для уязвимостей/сопротивлений) значение.

## Заклинания

Добавьте заклинательные записи:

- **Тип** — подготовленный, спонтанный, врождённый, фокусный
- **Традиция** — арканная, божественная, природная, мистическая
- **Класс кастера** — определяет слоты или пулы
- **Уровень кастера** — влияет на DC заклинаний и атаку
- **Заклинания** — добавьте из справочника

## Проверка здравомыслия

Кнопка **Проверить** запускает набор правил, которые предупреждают о:

- Статистиках значительно выше или ниже рекомендованных для уровня
- Отсутствующих атаках
- Некорректных сочетаниях (например, очень высокий AC + очень высокий урон одновременно)

Предупреждения — рекомендации, не ошибки. Существо сохраняется в любом случае.

## DC Вспоминания знаний

PathMaid автоматически рассчитывает DC для определения существа (Recall Knowledge) согласно правилам PF2e:

- Базовый DC по уровню
- Корректировка редкости (+2 необычное / +5 редкое / +10 уникальное)
- Рекомендуемые навыки для типа существа

## Использование в бою

1. В **Трекере боя** → **Добавить участника** → **Homebrew существо**
2. Выберите из списка созданных существ

Или из раздела **Конструктор** → нажмите **Добавить в бой** рядом с существом.

## Экспорт / копирование

Созданные существа хранятся локально. Для переноса между устройствами — используйте экспорт в JSON (кнопка **Экспорт** в конструкторе).
