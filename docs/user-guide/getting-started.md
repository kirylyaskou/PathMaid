# Начало работы

## Установка

Скачайте актуальный релиз со страницы [GitHub Releases](https://github.com/kirylyaskou/PathMaid/releases).

| Платформа | Файл |
|---|---|
| Windows | `PathMaid_X.Y.Z_x64-setup.exe` (установщик) или `PathMaid_X.Y.Z_x64.msi` |
| macOS | `PathMaid_X.Y.Z_x64.dmg` (Intel) или `PathMaid_X.Y.Z_aarch64.dmg` (Apple Silicon) |
| Linux | `PathMaid_X.Y.Z_amd64.AppImage` |
| Android | `PathMaid_X.Y.Z_universal.apk` |

Установите и запустите. При первом запуске приложение создаёт локальную базу данных.

## Первый запуск — пустая база

После установки база данных пуста. Без данных бестиарий, список заклинаний и предметов будут пустыми.

Есть два способа наполнить базу:

### Способ 1: Синхронизация с Foundry VTT

Если у вас установлен Foundry VTT с системой PF2e:

1. Откройте **Настройки** (иконка шестерёнки в боковой панели)
2. Найдите раздел **Синхронизация Foundry**
3. Укажите путь к папке данных Foundry (папка `Data`, содержащая `systems/` и `modules/`)
4. Нажмите **Синхронизировать**

Синхронизация импортирует существ, заклинания, предметы, опасности и действия из установленных паков PF2e.

### Способ 2: Ручной импорт

Если Foundry VTT не установлен, можно скачать отдельные JSON-паки системы PF2e и указать путь к ним. Актуальная версия системы: https://github.com/foundryvtt/pf2e

## Настройка отряда

После загрузки данных настройте параметры отряда:

1. Откройте **Главную страницу** (иконка домика)
2. Установите **Уровень отряда** — используется для расчёта XP и сложности встречи
3. Установите **Размер отряда** — по умолчанию 4

Эти параметры влияют на расчёт бюджета XP в конструкторе встреч.

| Мечи | Бой | Трекер инициативы |
| Дракон | Бестиарий | Поиск существ |
| Щит | Встречи | Сохранённые встречи |
| Звезда | Заклинания | Справочник заклинаний |
| Сумка | Предметы | Справочник предметов |
| Молния | Действия | Справочник действий PF2e |
| Черепаха | Опасности | Справочник опасностей |
| Маска | Состояния | Справочник состояний |
| Силуэт | Персонажи | Персонажи игроков |
| Молоток | Конструктор | Homebrew существа |
| Шестерёнка | Настройки | Синхронизация, настройки |
