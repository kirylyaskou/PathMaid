# Бестиарий и встречи

## Бестиарий

Раздел **Бестиарий** — поиск и просмотр существ из базы данных.

### Поиск и фильтрация

- **Строка поиска** — поиск по имени (полнотекстовый, поддерживает частичное совпадение)
- **Уровень** — фильтр по диапазону уровней
- **Размер** — крошечный / малый / средний / большой / огромный / колоссальный
- **Редкость** — обычный / необычный / редкий / уникальный
- **Источник** — фильтр по паку (Bestiary 1, 2, 3 и т.д.)

### Карточка существа

Нажмите на строку существа для открытия карточки. Карточка показывает:

- Уровень, размер, редкость, признаки
- Восприятие, языки
- Навыки, характеристики
- AC, HP, спасброски
- Скорость, иммунитеты/уязвимости/сопротивления
- Атаки
- Способности
- Заклинания
- Описание / лор

Кнопки в карточке:
- **Добавить в бой** — добавляет существо в текущий активный бой
- **Добавить во встречу** — добавляет в открытую встречу

### Локализация (RU)

Если для существа есть русский перевод, статблок отображается на русском. Значок показывает статус перевода: зелёный — есть RU, серый — только EN.

## Встречи

Раздел **Встречи** позволяет планировать и сохранять встречи.

### Создание встречи

1. Перейдите в **Встречи**
2. Нажмите **Новая встреча**
3. Задайте название

### Добавление существ

Используйте поиск внутри конструктора встречи:

- Введите имя существа
- Нажмите **+** рядом с результатом для добавления
- Укажите количество копий (несколько одинаковых существ)

Для каждого NPC можно выбрать вариант **Слабый / Нормальный / Элитный** — это изменит его вклад в XP и сложность встречи.

### Бюджет XP

PathMaid автоматически рассчитывает:

- **Общий XP** — сумма XP за всех участников
- **Сложность встречи** — ничтожная / лёгкая / умеренная / тяжёлая / смертельная
- **Бюджет по угрозе** — таблица с порогами XP для текущего уровня и размера отряда

Расчёт учитывает уровень отряда (из Главной страницы) и размер отряда.

### Опасности

В встречу можно добавлять **опасности** (hazards) из справочника опасностей. Опасности делятся на:

- **Простые** — дают XP как существо соответствующего уровня
- **Сложные** — дают XP за каждый раунд боя

### Запуск встречи

1. Нажмите **Начать бой** в встрече
2. Все участники переносятся в **Трекер боя**
3. После завершения боя состав встречи сохраняется, текущие HP — нет

### Импорт встречи

Встречи можно импортировать из внешних источников. Нажмите **Импорт** и вставьте JSON в поддерживаемом формате.

### Управление встречами

- **Переименовать** — нажмите на название
- **Дублировать** — создаёт копию встречи с тем же составом
- **Удалить** — безвозвратное удаление
