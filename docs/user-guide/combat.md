# Трекер боя

Трекер боя — главный инструмент для ведения сессии. Позволяет отслеживать инициативу, HP, состояния и эффекты всех участников боя.

## Создание боя

1. Перейдите в раздел **Бой**
2. Нажмите **Новый бой**
3. Добавьте участников одним из способов:
   - **Из бестиария** — найдите существо и нажмите «Добавить в бой»
   - **Персонаж игрока** — выберите из импортированных PC
   - **Произвольный** — введите имя вручную

## Инициатива

После добавления всех участников:

1. Введите значения инициативы для каждого участника
2. Нажмите **Начать бой** — список отсортируется по инициативе (убывание)
3. Текущий ход выделен подсветкой
4. Кнопка **Следующий ход** переходит к следующему участнику; по достижении последнего — новый раунд

Равные значения инициативы: NPC уступают игрокам; среди NPC порядок задаётся вручную (перетаскиванием).

## HP и урон

Кликните на участника для открытия панели управления:

- **Урон** — введите значение, нажмите минус или клавишу Enter
- **Лечение** — введите значение, нажмите плюс
- **Временные HP** — вводятся отдельным полем; поглощают урон первыми

Применение урона учитывает **IWR** (иммунитеты, уязвимости, сопротивления) если они заданы у существа.

### Слабый / Элитный вариант

Для NPC доступно переключение варианта: **Нормальный / Слабый / Элитный**.

Переключение автоматически пересчитывает:
- HP (±10/20/30 в зависимости от уровня)
- AC (±1/±2)
- Бонусы атаки и спасбросков (±1/±2)
- Урон (±2)
- Уровень существа (±1) — для расчёта XP

## Состояния

Для добавления состояния:

1. Кликните на участника
2. Нажмите **+ Состояние**
3. Выберите из списка

Для числовых состояний (испуганный, ошеломлённый и т.д.) введите значение.

### Особенности состояний

- **Умирающий** — автоматически отображает прогрессию смерти. При достижении dying 4 — существо мертво
- **Взамен умирающего** — при стабилизации dying заменяется на wounded (значение +1 к следующему dying)
- **Автоматические предоставления** — некоторые состояния автоматически добавляют другие (frightened → clumsy в определённых ситуациях)
- **Исключающие группы** — prone и некоторые другие состояния не могут стекаться с определёнными другими

## Эффекты заклинаний

Активные заклинания (Heroism, Bless, Shield и т.п.) отслеживаются как эффекты:

1. Нажмите **+ Эффект** в панели участника
2. Найдите нужный эффект по названию
3. Укажите оставшееся время (в раундах)

Эффекты автоматически применяют модификаторы к статистикам существа (бонусы AC, атаки, спасбросков и т.д.) согласно правилам PF2e.

Счётчик раундов уменьшается автоматически при переходе хода участника.

## Статблок существа

Кликните на имя NPC для открытия полного статблока. Статблок показывает:

- Базовые характеристики (STR/DEX/CON/INT/WIS/CHA)
- AC, HP, спасброски
- Восприятие, навыки
- Атаки и урон
- Способности
- Заклинания (если есть)
- IWR (иммунитеты, уязвимости, сопротивления)

Модификаторы от активных состояний и эффектов отображаются в реальном времени.

## Завершение боя

Нажмите **Завершить бой**. История боя не сохраняется — трекер очищается для следующего боя.

Если нужно сохранить встречу для повторного использования — используйте раздел **Встречи** (сохранение сохраняет состав, но не текущие HP/состояния).
