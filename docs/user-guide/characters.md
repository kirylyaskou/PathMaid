# Персонажи игроков

PathMaid позволяет импортировать персонажей PC из **Pathbuilder 2e** и использовать их в трекере боя.

## Импорт из Pathbuilder

### Шаг 1: Экспорт из Pathbuilder

1. Откройте персонажа в [Pathbuilder 2e](https://pathbuilder2e.com) (веб или мобильное приложение)
2. Перейдите в меню персонажа → **Export**
3. Выберите **Export JSON**
4. Сохраните JSON-файл

### Шаг 2: Импорт в PathMaid

1. Перейдите в раздел **Персонажи**
2. Нажмите **Импортировать персонажа**
3. Вставьте содержимое JSON-файла или загрузите файл
4. Нажмите **Импортировать**

Персонаж появится в списке с именем, классом, уровнем и расой.

## Карточка персонажа

Нажмите на персонажа для просмотра:

- Имя, класс, уровень, раса
- Характеристики (STR/DEX/CON/INT/WIS/CHA)
- HP (максимальный)
- AC, спасброски
- Навыки
- Заклинания (если есть)
- Способности

## Использование в бою

1. В **Трекере боя** → **Добавить участника** → **Персонаж игрока**
2. Выберите нужного персонажа из списка
3. Персонаж добавляется с правильными HP и статистикой

HP отслеживается отдельно в трекере боя — изменения не сохраняются обратно в Pathbuilder.

## Обновление персонажа

При повышении уровня:

1. Экспортируйте обновлённый JSON из Pathbuilder
2. В PathMaid нажмите **...** рядом с персонажем → **Обновить**
3. Вставьте новый JSON

Старые данные заменяются новыми.

## Удаление персонажа

Нажмите **...** рядом с персонажем → **Удалить**. Персонаж удаляется из базы данных PathMaid (не из Pathbuilder).

## Заметки

Для каждого персонажа можно вести заметки — открытое текстовое поле без форматирования. Заметки сохраняются локально и не связаны с Pathbuilder.
