# Plan 22-01: Creature Inventory Display in Stat Block

**Phase:** 22 — Creature Inventory Display
**Requirements:** CRINV-01, CRINV-02
**Status:** Not started

## Goal

Show a creature's carried items (non-strike equipment) in the stat block under an "Equipment" section. Data comes from `creature_items` table populated in Phase 20.

## Changes

- `src/entities/creature/model/fetchStatBlock.ts`:
  - Import `getCreatureItems` from `@/shared/api`
  - Call `getCreatureItems(creatureId)` alongside existing spellcasting fetch
  - Extend returned `CreatureStatBlockData` to include `equipment?: CreatureItemRow[]`

- `src/entities/creature/model/types.ts`:
  - Add `equipment?: CreatureItemRow[]` to `CreatureStatBlockData`
  - Import `CreatureItemRow` from `@/shared/api`

- `src/entities/creature/ui/CreatureStatBlock.tsx`:
  - Add `EquipmentBlock` component (local, below SpellcastingBlock):
    - Header: "Equipment" with backpack icon
    - List of items: name, quantity (if >1), bulk, damage formula or AC bonus, traits
    - Grouped by type: weapons first, then armor, then consumables, then equipment/misc
    - Collapsed by default (Collapsible, same as SpellcastingBlock)
    - Only rendered when `creature.equipment && creature.equipment.length > 0`

## Notes

- `CreatureItemRow` imported from `@/shared/api` — same type used in items catalog
- Weapons shown as "Longsword (1d8 slashing)" — formula from damage_formula column
- Armor shown as "Breastplate (AC +4)" — from ac_bonus column
- Consumables shown with uses if uses_max > 1: "Healing Potion ×3"
- No encounter context needed here — this is always the base creature inventory
- Per-encounter editing is Phase 23 scope
